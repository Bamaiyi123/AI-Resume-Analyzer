# AI Resume Analyzer

## Setup Instructions

1. Install Python (3.9+ recommended)
2. Install dependencies:

   pip install -r requirements.txt

3. Run the app:

   streamlit run app.py

4. Open in browser:
   http://localhost:8501

## Features
- Upload PDF resume
- Paste job description
- Get AI-powered analysis (match score, missing skills, suggestions)

## Notes
- Requires OpenAI API key
